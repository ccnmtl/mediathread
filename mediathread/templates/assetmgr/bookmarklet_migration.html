Bookmarklet Migration

It looks like you've run into a bookmarklet error caused by either the website you were on, or your web browser. This happened because there's a new web security standard that doesn't work well with bookmarklets.

Here are two different ways you can get around this limitation:

  • Try another web browser. If you're using Google Chrome, try Firefox or Safari.
  • Use Mediathread's browser extension to collect media instead of the bookmarklet. The Mediathread extension is available for Chrome, Safari, and Firefox. You can install it using the buttons below.
